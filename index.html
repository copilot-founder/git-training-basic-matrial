Quick Git FAQ

What is Git commit?

Like Ctrl+S but creates a snapshot you can return to anytime.

Basic Git commands?

git add = stage files | git commit = save snapshot | git push = upload to GitHub | git pull = download updates

What is a branch?

A copy of your project to experiment without breaking the original.

What is Pull Request?

Asking "Can I merge my changes?" - like showing work before submitting.

Need to install Git?

No! Practice at gitlearn - browser-based Git simulator.
