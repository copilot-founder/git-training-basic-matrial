## 👋 Welcome to Git Training Discussions!

This is your space to ask, share, and learn from real Git & GitHub scenarios — no question is too basic.

### 💬 What you can do here

- 🙋 **Ask anything** — stuck on a merge conflict? unsure when to branch? just ask
- 💡 **Share your scenario** — tell us a real situation where Git saved (or confused) you
- 🧪 **Share experiments** — tried something new with branches or workflows? show us
- 🤝 **Help others** — if you know the answer, jump in

### 🎯 Popular topics to get you started

| Topic | Example |
|-------|---------|
| Branching | "When should I create a new branch?" |
| Merge conflicts | "I got a conflict — what do I do?" |
| Pull Requests | "How do I review someone's PR?" |
| Undo mistakes | "I committed to wrong branch — help!" |
| Daily workflow | "What does your team's Git routine look like?" |
| SVN → Git | "Coming from SVN, what's different?" |
